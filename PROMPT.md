# 🌙 MoonVPN AI Prompts راهنمای دستورات هوش مصنوعی

این فایل شامل مجموعه‌ای از پرامپت‌های ساختاریافته برای تعامل با هوش مصنوعی در پروژه MoonVPN است. از این دستورات برای هدایت هوش مصنوعی در مراحل مختلف توسعه استفاده کنید.

## 🎯 حالت برنامه‌ریزی (Plan Mode)

### شروع برنامه‌ریزی سیستمی
```
plan
با توجه به @scratchpad، @memories و @project-requirements، لطفاً برنامه‌ریزی جامعی برای [هدف/تسک] انجام دهید. نیازمندی‌ها، سوالات کلیدی، مراحل اجرا و تأثیر آن بر کل سیستم را مشخص کنید.
```

### تحلیل معماری و وابستگی‌ها
```
plan
لطفاً تحلیل کاملی از تأثیر [تغییر/ویژگی جدید] بر معماری موجود و وابستگی‌های سیستم انجام دهید. به @lessons-learned برای درس‌های گذشته و @memories برای تجربیات مشابه مراجعه کنید.
```

### بررسی امکان‌پذیری و ریسک
```
plan
آیا [ویژگی/تغییر] با توجه به معماری موجود و @project-requirements امکان‌پذیر است؟ لطفاً ریسک‌ها، چالش‌ها، و تأثیر آن بر امنیت، عملکرد و مقیاس‌پذیری سیستم را بررسی کنید.
```

### تحلیل زنجیره‌ای مشکلات
```
plan
مشکل [توضیح مشکل] در [بخش مربوطه] رخ داده است. لطفاً با استفاده از تحلیل زنجیره‌ای فکر و مراجعه به @lessons-learned، ریشه‌یابی کرده و راه‌حل‌های سیستمی ارائه دهید.
```

## ⚡ حالت اجرا (Agent Mode)

### پیاده‌سازی با حفظ یکپارچگی
```
agent
بر اساس برنامه‌ریزی انجام شده و با توجه به @project-requirements، لطفاً [تغییر/ویژگی] را با حفظ یکپارچگی سیستم پیاده‌سازی کنید. اطمینان حاصل کنید که تمام استانداردهای مستندسازی و تست رعایت شده‌اند.
```

### بهینه‌سازی و بهبود
```
agent
لطفاً [بخش/ویژگی] را با هدف بهبود [عملکرد/امنیت/مقیاس‌پذیری] بهینه‌سازی کنید. از @lessons-learned برای اجتناب از مشکلات مشابه استفاده کنید.
```

### مستندسازی جامع
```
agent
لطفاً مستندات کامل برای [تغییر/ویژگی] را طبق استانداردهای @project-requirements ایجاد کنید. این شامل مستندات فنی، راهنمای کاربر، و به‌روزرسانی CHANGELOG می‌شود.
```

### ثبت درس‌آموخته‌ها
```
لطفاً درس‌های آموخته شده از [تجربه/چالش] اخیر را در @lessons-learned ثبت کنید. این شامل چالش‌ها، راه‌حل‌ها و بهترین شیوه‌ها می‌شود.
```

## 🛠️ پرامپت‌های توسعه و نگهداری

### بررسی کیفیت کد
```
لطفاً کیفیت کد در [مسیر/بخش] را با توجه به استانداردهای پروژه بررسی کنید. موارد نیازمند بهبود، الگوهای تکراری و فرصت‌های بهینه‌سازی را شناسایی کنید.
```

### مدیریت تکنیکال دبت
```
لطفاً [بخش/ماژول] را برای شناسایی تکنیکال دبت بررسی کنید. راهکارهایی برای بهبود تدریجی کد، معماری و مستندات ارائه دهید.
```

لطفاً [ویژگی/بخش] را از نظر امنیتی بررسی کنید. آسیب‌پذیری‌های احتمالی، نقاط ضعف و راهکارهای بهبود را شناسایی کنید.
```

### بهینه‌سازی عملکرد
```
لطفاً عملکرد [بخش/ویژگی] را تحلیل کنید. گلوگاه‌ها، فرصت‌های بهینه‌سازی و راهکارهای بهبود را شناسایی کنید.
```

```
لطفاً متون [بخش/ویژگی] را بررسی و بهینه کنید. اطمینان حاصل کنید که ترجمه‌ها دقیق، دوستانه و حرفه‌ای هستند و از ایموجی‌های مناسب استفاده می‌کنند.
```

### بهینه‌سازی تجربه کاربری

```

لطفاً یک استراتژی جامع تست برای [ویژگی/تغییر] طراحی کنید. این باید شامل تست‌های واحد، یکپارچگی و E2E باشد.

### تحلیل وضعیت سیستم
```
لطفاً وضعیت کلی سیستم را با توجه به [معیارها] تحلیل کنید. نقاط قوت، ضعف و فرصت‌های بهبود را شناسایی کنید.
```

### گزارش پیشرفت
```
لطفاً گزارشی جامع از پیشرفت [پروژه/بخش] با توجه به @scratchpad و @memories ارائه دهید. این شامل دستاوردها، چالش‌ها و برنامه‌های آینده می‌شود.
```

## 🚀 نکات مهم برای استفاده از پرامپت‌ها

1. **مدیریت حافظه و دانش**
   - همیشه به @memories و @lessons-learned برای تجربیات گذشته مراجعه کنید
   - تجربیات و درس‌های جدید را ثبت کنید
   - از تکرار اشتباهات گذشته اجتناب کنید

2. **رعایت استانداردها**
   - همیشه @project-requirements را مرور کنید
   - استانداردهای کدنویسی و مستندسازی را رعایت کنید
   - یکپارچگی سیستم را حفظ کنید

3. **مدیریت تغییرات**
   - تأثیر تغییرات بر کل سیستم را بررسی کنید
   - تغییرات را در @scratchpad ثبت کنید
   - CHANGELOG را به‌روز نگه دارید

4. **کیفیت و تست**
   - برای هر تغییر، تست مناسب بنویسید
   - پوشش تست را حفظ کنید
   - کیفیت کد را بررسی کنید

5. **مستندسازی**
   - تمام تغییرات را مستند کنید
   - مستندات را به‌روز نگه دارید
   - از کامنت‌های مناسب استفاده کنید

6. **امنیت و عملکرد**
   - امنیت را در اولویت قرار دهید
   - عملکرد را بهینه کنید
   - مقیاس‌پذیری را در نظر بگیرید

7. **تجربه کاربری**
   - متون فارسی جذاب بنویسید
   - از ایموجی‌های مناسب استفاده کنید
   - تجربه کاربری را بهبود دهید
