# 🌙 MoonVPN

<div dir="rtl">

## معرفی

MoonVPN یک سیستم مدیریت، اتوماسیون و فروش خدمات VPN است که به شما امکان می‌دهد چندین پنل x-ui را از طریق یک پلتفرم متمرکز و با استفاده از بات تلگرام مدیریت کنید.

### ویژگی‌ها

- 🚀 **مدیریت چندین پنل**: همه پنل‌های X-UI را از یک مکان مدیریت کنید
- 🤖 **بات تلگرام**: مدیریت سیستم و فروش خودکار سرویس از طریق تلگرام
- 🔔 **اعلان‌ها**: دریافت اعلان‌های مربوط به وضعیت سرویس‌ها، تراکنش‌ها و خطاها
- 💳 **پرداخت خودکار**: پرداخت خودکار با درگاه‌های مختلف
- 📊 **داشبورد مدیریت**: نظارت و کنترل پنل‌ها، کاربران و تراکنش‌ها
- 📈 **آمار**: گزارش‌های مصرف ترافیک، کاربران و درآمد
- 🔄 **بک‌آپ خودکار**: پشتیبان‌گیری خودکار از داده‌ها و تنظیمات
- 🧪 **تست خودکار**: تست سلامت سرویس‌ها و سرعت سرورها

## نیازمندی‌ها

- سیستم عامل: Linux (Ubuntu 20.04+, Debian 11+)
- Docker و Docker Compose
- Python 3.8+
- پنل x-ui نصب شده (نسخه 3.0+)
- دسترسی به API تلگرام برای بات

## نصب سریع

برای نصب MoonVPN، دستورات زیر را اجرا کنید:

```bash
# کلون کردن مخزن
git clone https://github.com/yourusername/moonvpn.git
cd moonvpn

# اجرای اسکریپت نصب
./scripts/install.sh
```

یا می‌توانید از دستور زیر برای نصب با یک خط استفاده کنید:

```bash
curl -fsSL https://raw.githubusercontent.com/yourusername/moonvpn/main/scripts/get.sh | bash
```

## راه‌اندازی

بعد از نصب، می‌توانید از دستور `moonvpn` برای مدیریت سرویس‌ها استفاده کنید:

```bash
# راه‌اندازی سرویس‌ها
moonvpn start

# مشاهده وضعیت سرویس‌ها
moonvpn status

# مشاهده لاگ‌ها
moonvpn logs [سرویس]

# پشتیبان‌گیری
moonvpn backup

# راه‌اندازی مجدد سرویس‌ها
moonvpn restart

# متوقف کردن سرویس‌ها
moonvpn stop

# بررسی سلامت سیستم
moonvpn healthcheck
```

## پیکربندی

پیکربندی اصلی در فایل `.env` قرار دارد. این فایل به صورت خودکار در هنگام نصب از روی `.env.example` ساخته می‌شود. شما باید این فایل را ویرایش کرده و متغیرهای محیطی مورد نیاز را تنظیم کنید.

### متغیرهای محیطی مهم

```env
# تنظیمات API
API_PORT=8000
API_SECRET_KEY=your_very_secure_secret_key

# تنظیمات دیتابیس
MYSQL_HOST=moonvpn_db
MYSQL_PORT=3306
MYSQL_DATABASE=moonvpn
MYSQL_USER=moonvpn
MYSQL_PASSWORD=secure_password

# تنظیمات بات تلگرام
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
TELEGRAM_ADMIN_IDS=123456789,987654321

# تنظیمات پنل
PANEL1_URL=https://your-panel-url.com
PANEL1_USERNAME=admin
PANEL1_PASSWORD=admin_password
```

## حل مشکلات متداول

### بات تلگرام کار نمی‌کند

1. مطمئن شوید که توکن بات درست است و بات فعال است.
2. لاگ بات را بررسی کنید: `moonvpn logs bot`
3. مطمئن شوید که متغیر `TELEGRAM_BOT_TOKEN` در فایل `.env` به درستی تنظیم شده است.

### نمی‌توانم به API وصل شوم

1. مطمئن شوید که پورت API در فایروال باز است.
2. لاگ API را بررسی کنید: `moonvpn logs api`
3. مطمئن شوید که سرویس API در حال اجراست: `moonvpn status`

### ارور اتصال به دیتابیس

1. مطمئن شوید که اطلاعات دیتابیس در فایل `.env` درست است.
2. بررسی کنید که سرویس دیتابیس در حال اجراست: `moonvpn status`
3. لاگ‌های دیتابیس را بررسی کنید: `moonvpn logs db`

## پشتیبان‌گیری و بازیابی

برای پشتیبان‌گیری دستی:

```bash
moonvpn backup
```

این دستور یک فایل پشتیبان در پوشه `backups/` ایجاد می‌کند.

برای بازیابی از پشتیبان:

```bash
moonvpn restore backups/moonvpn_backup_YYYY-MM-DD_HH-MM-SS.tar.gz
```
