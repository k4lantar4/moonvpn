# مون وی‌پی‌ان - ربات تلگرام و داشبورد وب

مون وی‌پی‌ان یک سیستم جامع مدیریت وی‌پی‌ان با رابط ربات تلگرام و داشبورد وب برای فروش و مدیریت حساب‌های وی‌پی‌ان است. این سیستم با پنل‌های 3x-UI ادغام شده تا تجربه‌ای یکپارچه برای مدیران و کاربران فراهم کند.

## ساختار پروژه

پروژه به بخش‌های اصلی زیر سازماندهی شده است:

- **bot/**: کد ربات تلگرام (پایتون)
- **backend/**: API و منطق بک‌اند مبتنی بر Django
- **frontend/**: داشبورد React با پنل‌های مدیریتی
- **docker/**: فایل‌های پیکربندی Docker
- **scripts/**: ابزارهای خط فرمان برای مدیریت سمت سرور
- **nginx/**: پیکربندی Nginx برای استقرار در محیط تولید

## ویژگی‌ها

- ربات تلگرام برای تعامل کاربر
- داشبورد وب برای مدیریت
- ادغام با پنل 3x-UI برای مدیریت حساب‌های وی‌پی‌ان
- پردازش پرداخت (کارت به کارت و زرین‌پال اختیاری)
- مدیریت کاربر و پیگیری اشتراک
- متعادل‌سازی بار سرور و نظارت
- سیستم کد تخفیف
- پشتیبانی از چند زبان (فارسی و انگلیسی)
- تجزیه و تحلیل و گزارش‌دهی دقیق

## نیازمندی‌ها

- Docker و Docker Compose
- پایتون 3.11+
- Node.js 18+ (برای توسعه فرانت‌اند)
- PostgreSQL 15+
- Redis 7+
- سرور پنل 3x-UI

## شروع سریع

1. کلون کردن مخزن:
   ```bash
   git clone https://github.com/k4lantar4/moonvpn.git
   cd moonvpn
   ```

2. کپی فایل محیطی نمونه و پیکربندی آن:
   ```bash
   cp .env.example .env
   # ویرایش .env با پیکربندی خود
   ```

3. راه‌اندازی سرویس‌ها با استفاده از Docker Compose:
   ```bash
   docker-compose up -d
   ```

4. دسترسی به سرویس‌ها:
   - API بک‌اند: http://localhost:8000/api/v1/
   - داشبورد فرانت‌اند: http://localhost:8080/
   - ربات تلگرام: شروع گفتگو با ربات خود در تلگرام

## راه‌اندازی محیط توسعه

### بک‌اند

1. ایجاد محیط مجازی:
   ```bash
   python -m venv venv
   source venv/bin/activate  # در ویندوز: venv\Scripts\activate
   ```

2. نصب وابستگی‌ها:
   ```bash
   pip install -r requirements.txt
   ```

3. اجرای مهاجرت‌ها:
   ```bash
   cd backend
   python manage.py migrate
   ```

4. راه‌اندازی سرور توسعه:
   ```bash
   python manage.py runserver
   ```

### ربات

1. با فعال بودن محیط مجازی، اجرا کنید:
   ```bash
   cd bot
   python main.py
   ```

### فرانت‌اند

1. نصب وابستگی‌ها:
   ```bash
   cd frontend
   npm install
   ```

2. راه‌اندازی سرور توسعه:
   ```bash
   npm start
   ```

## پیکربندی

پروژه از متغیرهای محیطی برای پیکربندی استفاده می‌کند. متغیرهای کلیدی شامل:

- `TELEGRAM_BOT_TOKEN`: توکن ربات تلگرام شما
- `ADMIN_IDS`: لیست شناسه‌های تلگرام مدیران، جدا شده با کاما
- `X_UI_PANEL_URL`: آدرس پنل 3x-UI شما
- `X_UI_PANEL_USERNAME`: نام کاربری برای پنل 3x-UI
- `X_UI_PANEL_PASSWORD`: رمز عبور برای پنل 3x-UI
- `DB_NAME`، `DB_USER`، `DB_PASSWORD`: اعتبارنامه‌های پایگاه داده
- `CARD_NUMBER`، `CARD_HOLDER`: جزئیات کارت پرداخت

برای لیست کامل گزینه‌های پیکربندی، به `.env.example` مراجعه کنید.
