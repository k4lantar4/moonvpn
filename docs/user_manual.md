## Introduction | مقدمه
MoonVPN is a comprehensive VPN service management system that provides secure and reliable VPN connections through an easy-to-use Telegram bot interface.

MoonVPN یک سیستم مدیریت سرویس VPN است که از طریق یک رابط کاربری ساده در تلگرام، اتصالات VPN امن و قابل اعتماد را ارائه می‌دهد.

### Features | ویژگی‌ها
- Easy account management through Telegram | مدیریت آسان حساب از طریق تلگرام
- Multiple server locations | چندین موقعیت سرور
- Secure payment options | گزینه‌های پرداخت امن
- 24/7 support | پشتیبانی ۲۴/۷
- Automatic configuration | پیکربندی خودکار

## Getting Started | شروع کار

### 1. Finding the Bot | پیدا کردن ربات
1. Open Telegram | تلگرام را باز کنید
2. Search for @MoonVPNBot | جستجوی @MoonVPNBot
3. Start the bot by clicking Start | با کلیک روی Start، ربات را شروع کنید

### 2. Registration | ثبت نام
1. Send /start command | دستور /start را ارسال کنید
2. Enter your phone number | شماره تلفن خود را وارد کنید
3. Enter verification code | کد تایید را وارد کنید
4. Complete profile information | اطلاعات پروفایل را تکمیل کنید

## Using the Telegram Bot | استفاده از ربات تلگرام

### Available Commands | دستورات موجود
- /start - Start the bot | شروع ربات
- /buy - Purchase VPN plans | خرید پلن‌های VPN
- /status - Check account status | بررسی وضعیت حساب
- /help - Get help | دریافت راهنمایی
- /settings - Manage settings | مدیریت تنظیمات

### Navigation | پیمایش
Use the provided keyboard buttons for easy navigation through different sections.
از دکمه‌های صفحه کلید ارائه شده برای پیمایش آسان در بخش‌های مختلف استفاده کنید.
